These are detailed figures so may take a little time to load. Taller blocks correspond to higher carbon footprints. Countries with larger populations are assigned more blocks. Colours correspond to how rich the country is in 1990, with poorer countries shaded red and richer countries shaded green. Countries are ordered according to mean income level in each year, while holding colours fixed from the 1990 levels. Therefore with these figures we can see inequality within and across countries over time as well as changes in average income. Try hovering over a block to learn more!

Source: World Inequality Database

Countries with population size under 750,000 are omitted.
